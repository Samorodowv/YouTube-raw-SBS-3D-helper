# Store Asset Checklist

## Required Before Submission

- `icons/icon128.png`
- At least one real Chrome screenshot showing the extension active on a supported YouTube page
- Small promotional tile `440x280`

## Recommended Screenshots

- Player with `SBS ON` visible on the control bar
- Same page with `SBS OFF` for comparison
- Supported YouTube watch page in theater mode

## Capture Guidance

- Use Chrome on Windows with the unpacked extension loaded
- Use a real supported video, preferably `https://www.youtube.com/watch?v=tbouGUCUZGo`
- Avoid personal account information in the capture
- Keep screenshots accurate to the current UI and extension behavior
