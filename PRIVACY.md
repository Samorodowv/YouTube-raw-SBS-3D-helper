# Privacy Policy

YouTube raw SBS/3D helper does not collect, transmit, sell, or share personal data.

## What The Extension Accesses

- The extension runs on `https://www.youtube.com/*` pages.
- It reads the YouTube player DOM to detect whether a supported raw-video plus canvas layout is present.
- It stores one local preference in browser extension storage: whether the SBS toggle is enabled.

## What The Extension Does Not Do

- It does not send browsing history, video URLs, account information, cookies, or player state to any server.
- It does not inject remote code.
- It does not use analytics, ads, trackers, or external APIs.
- It does not modify pages outside YouTube.

## Data Storage

- `enabled`: local browser extension storage only, used to remember whether SBS mode should default to on.
