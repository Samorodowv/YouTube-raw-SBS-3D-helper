Support

Bug reports and reproducible cases

The fastest support path is a GitHub issue with a working YouTube URL and a short description of what the player did.

Open an issue Back to site

Include in every report

  • The full YouTube URL
  • Whether the video is expected to be SBS or top-bottom
  • Your Chrome version
  • Whether the problem happens on reload or after in-site navigation
  • A console screenshot if there is an error

Known failure modes

  • YouTube changed player selectors or control markup.
  • The current page does not expose a raw video plus processed canvas pair.
  • Another extension is modifying the YouTube player UI or DOM.

Scope

This extension only changes YouTube player presentation locally in the browser. It does not restore missing 3D metadata on YouTube and it does not alter the server-side video stream.
