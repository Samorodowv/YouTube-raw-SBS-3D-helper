Chrome Extension

YouTube raw SBS/3D helper

Restore the raw SBS or top-bottom 3D stream on supported YouTube watch pages by hiding the processed canvas layer and surfacing the underlying video element.

Chrome-first Local player-only No remote code No data collection
Support Privacy policy GitHub

What it does

On supported YouTube watch pages, the extension detects the active player, finds the raw HTML5 video element and the processed canvas layer, and switches the local presentation back to the raw stereo stream.

What it does not do

It does not download, re-encode, or upload video. It does not change YouTube data on the server. It only adjusts the local player DOM in your browser.

Current scope

The extension is designed for historical YouTube 3D uploads where the raw SBS or top-bottom stream is still present in the page player structure.

How to use it

  • Install the unpacked or published Chrome extension.
  • Open a supported YouTube watch page.
  • The extension enables SBS mode automatically when it detects a compatible player structure.
  • Use the in-player SBS ON or SBS OFF control to override the current page state.
