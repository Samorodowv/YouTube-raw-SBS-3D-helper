Privacy Policy

No collected or transmitted user data

YouTube raw SBS/3D helper does not collect, transmit, sell, or share personal data. It operates locally on supported YouTube pages and stores one local preference in extension storage.

Back to site GitHub

What the extension accesses

  • It runs on https://www.youtube.com/* pages.
  • It reads the YouTube player DOM to detect whether a supported raw-video plus canvas layout is present.
  • It stores one local setting to remember whether SBS mode should default to on.

What the extension does not do

  • It does not send browsing history, video URLs, account information, or cookies to any server.
  • It does not inject remote code.
  • It does not use analytics, ads, trackers, or external APIs.
  • It does not run on pages outside YouTube.

Stored data

enabled: local browser extension storage only, used to remember whether SBS mode should be enabled by default.
